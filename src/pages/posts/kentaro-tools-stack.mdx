# kentaro.tools Stack

## Stack
- Framework: [Next.js](https://nextjs.org/)
- DB + Auth: [Supabase](https://supabase.com/)
- Storage: [Cloudflare R2](https://www.cloudflare.com/developer-platform/r2/)
- Payments: [Paddle](https://www.paddle.com/)
- Hosting: [Vercel](https://vercel.com)
- UI: [Radix UI](https://www.radix-ui.com/) + [Tailwind CSS](https://tailwindcss.com/)
- State Management: [Zustand](https://github.com/pmndrs/zustand) + [Tanstack Query](https://tanstack.com/query/latest)
